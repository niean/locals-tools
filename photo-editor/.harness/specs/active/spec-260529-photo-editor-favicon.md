# PhotoEditor favicon 设计说明

## 目标
为默认页面补全一个与产品定位一致的 favicon：蓝色主题、个人工具导向、在浏览器标签页小尺寸下保持高识别度。

## 背景
当前 [photo-editor/index.html](photo-editor/index.html) 已引用 `favicon.svg`，但现有图标风格偏线框通用图形，缺少更明确的“个人工具”语义，也不够贴合产品轻量、本地、即开即用的定位。

## 设计结论
采用已确认的 A 方案：亮蓝底圆角方形徽章 + 白色极简工具箱符号 + 一笔蓝色斜线作为编辑语义补充。

## 视觉原则
- 蓝色主题：使用亮蓝主色，传达轻快、高效、工具属性
- 个人工具：主体图形优先表达工具箱，而非复杂图片语义
- 扁平几何：避免渐变、阴影、细碎笔画，保证 16x16 可读性
- 小尺寸优先：轮廓大于细节，优先标签页可识别性

## 图形组成
1. 外层为亮蓝色圆角方形底板，作为应用徽章轮廓
2. 中间为白色工具箱主体，形状简洁、留白充足
3. 右上加入一笔短斜线，弱化为“编辑动作”提示，不与工具箱主语义竞争
4. 工具箱内部保留单个蓝色圆点，增加层次并避免完全空心

## 文件与集成
- 修改 `favicon.svg`
- 保持 `index.html` 中现有 `<link rel="icon" href="favicon.svg" type="image/svg+xml">` 不变

## 非目标
- 不新增多套 favicon 规格
- 不引入 PNG/ICO 生成流程
- 不调整页面其他品牌视觉元素

## 验收标准
- 页面继续使用 `favicon.svg` 作为站点图标来源
- favicon 为亮蓝主题
- favicon 的主语义为“个人工具”
- 在小尺寸下仍能辨认出圆角徽章和工具主体
